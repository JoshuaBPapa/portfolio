Bug Tracker

A web app built to keep a track of bugs in projects by creating tickets. A user can create a team and add other users to it. These created users can then be assigned to tickets with different levels of priority and progress.

Technology

React was used to create the frontend due to multiple components being reused. A custom hook, useAxios, was also created to handle API requests and JWTs. The REST API was built using Node/Express and MySQL. The API is hosted on Heroku and the database on Amazon RDS.
